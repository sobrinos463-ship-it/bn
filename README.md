# Project Context: Actúa como un Arquitecto de Software Senior y un Product Manager experto en aplicaciones de Fitness e Inteligencia Artificial. Quiero desarrollar una aplicación móvil de gimnasio revolucionaria cuyo núcleo sea un "Coach IA" totalmente integrado y proactivo.

**Visión del Producto:**
La aplicación no es solo un registro de ejercicios; es un entrenador personal inteligente de bolsillo. El usuario no elige rutinas de una lista estática; la IA genera, adapta y gestiona el plan basándose totalmente en el objetivo del usuario (ej. hipertrofia, fuerza, pérdida de grasa, resistencia) y su feedback diario.

**Requisitos Funcionales Clave:**

1.  **Onboarding Inteligente:** Un sistema de entrevista inicial donde la IA analice edad, peso, experiencia, lesiones, días disponibles y equipamiento para crear el "Plan Maestro".
2.  **El Coach IA (Core):**
    * Debe explicar el "por qué" de cada ejercicio.
    * Debe tener capacidad de chat natural (NLP) para dudas ("Me duele la rodilla, ¿qué hago?", "No tengo esta máquina").
    * **Adaptabilidad en Tiempo Real:** Si el usuario falla un levantamiento o se siente cansado, la IA debe reestructurar el resto del entrenamiento al instante (bajar peso, cambiar repeticiones).
3.  **Sobrecarga Progresiva Automática:** La IA debe calcular cuándo subir los pesos o repeticiones basándose en el rendimiento de la sesión anterior, sin que el usuario tenga que adivinar.
4.  **Feedback Visual (Opcional pero deseable):** Posibilidad de analizar la técnica mediante visión por computadora (cámara del móvil).

**Lo que necesito que me entregues ahora:**

1.  **Arquitectura Técnica:** Recomienda el Stack Tecnológico ideal (Frontend, Backend, Base de Datos y qué modelo de LLM/API de IA usarías para el coach).
2.  **Modelo de Datos:** Un esquema preliminar de la base de datos para manejar Usuarios, Rutinas, Ejercicios, Historial y la "Memoria" de la IA.
3.  **User Flow del "Entrenamiento Diario":** Paso a paso de cómo interactúa el usuario con la IA desde que abre la app hasta que termina de entrenar.
4.  **Prompt del Sistema (System Prompt):** Escribe el prompt inicial que yo debería darle a la API de la IA (como OpenAI o Anthropic) para que se comporte como este entrenador personal empático, estricto pero motivador.
